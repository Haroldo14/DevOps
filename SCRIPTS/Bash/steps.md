# Script de sauvegarde automatique avec Windows + git Bash

## Creation du script backup.sh
```bash
vim backup.sh
```

Test manuel script
```bash
./backup.sh
```

## Automatisation de l'execution
### Sous Windows
Ouvrir Task Scheduler via 'windows+r'
```bash
taskschd.msc
```
